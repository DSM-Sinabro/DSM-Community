# DSM Community

## 개요
그 동안 대덕SW마이스터고등학교 관련 여러 시스템들이 개발되었지만 학생들이 서로 소통하고 질의응답을 가질수있는 시스템은 없었습니다.  
그래서 저희는 재학생과 재학생, 졸업생과 재학생, 재학생과 선생님이 소통 할 수있는 공간을 만들고자 DSM Community를 만들게 되었습니다.
